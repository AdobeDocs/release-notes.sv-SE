#### Korrigeringar

* Ett problem har korrigerats där [!UICONTROL People] - och [!UICONTROL Offers] bastjänsterna inte kunde nås från huvudmenyn på Experience Cloud. (AN-184294)
* Korrigerade ett problem med den vänstra rälen i [!UICONTROL Analysis Workspace] oscillering mellan att ha en rullningslist och att inte ha någon rullningslist, vilket orsakade en flödningseffekt. (AN-183904)
* Åtgärdade problem med felrapportering. Du kommer att börja se mer specifika felmeddelanden i stället för bara den röda felindikatorn. Mer specifikt bör den hjälpa dig att förstå när problemet har orsakats av hög belastning, av ett fel eller av att skapa en rapportbegäran som är för komplex. (AN-184135) [Mer...](https://docs.adobe.com/content/help/en/analytics/analyze/analysis-workspace/optimizing-performance.html)
* Ett problem som gjorde att det inte gick att hämta bortfallsrapporter i `.pdf/.xls/.rtf` format har korrigerats. (AN-183165)
* Problem med att logga in via Experience Cloud och växla till olika Experience Cloud-lösningar eller till ett annat inloggningsföretag har åtgärdats. (AN-183376)
* Korrigerade ett problem med resursöverföring av schemalagda projekt som inte fungerade korrekt. Grupper hanteras i [!UICONTROL Admin Console] nuet så att vi inte kopierar dem mellan användare när vi överför resurser längre. (AN-183751)
* Ett problem har korrigerats med borttagning av schemalagda rapporter vars ägare har tagits bort. Från och med nu skickas ett meddelande till administratören (som utförde borttagningsåtgärden) när schemaägaren inte längre finns. (AN-181000)

## [!DNL Campaign] {#ac}

Adobe Campaign är ett intuitivt och automatiserat sätt att leverera personliga meddelanden i alla marknadsföringskanaler, både online och offline. Nu kan ni förutse vad era kunder vill ha med upplevelser som bestäms av deras vanor och önskemål.

### Adobe Campaign Classic

* [Campaign Classic 19.1.4-uppdatering](https://docs.campaign.adobe.com/doc/AC/en/RN.html#9032) - build 9032
* [Campaign Classic 19.1.5-uppdatering](https://docs.campaign.adobe.com/doc/AC/en/RN.html#9033) - build 9033

### Adobe Campaign [!UICONTROL Control Panel]

Vi har lagt till nya funktioner för administratörsanvändare så att IP-adresser kan ansluta till instanser i Campaign Classic.
Dessutom kan administratörsanvändare nu se listan över instanser i och berättigande till uppgraderingar av builds. Se den [särskilda dokumentationen](https://helpx.adobe.com/se/campaign/kb/control-panel-instance-settings.html) för mer information.

Vi har lagt till nya funktioner för administratörsanvändare som nu kan få meddelanden innan SSL-certifikat för deras domäner upphör att gälla. Se den [detaljerade dokumentationen](https://helpx.adobe.com/se/campaign/kb/control-panel-subdomains-certificates.html) för mer information.

Administratörsanvändare kan nu även ta bort SSH-nycklar som har lagts till för att komma åt SFTP-servrar.

Observera att det [!UICONTROL Control Panel] finns både för Adobe Campaign Classic- och Adobe Campaign Standard-kunder på AWS. Det krävs inga uppgraderingar för att få tillgång till [!UICONTROL Control Panel].
